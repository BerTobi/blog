Settings

Your token is stored only in this browser (localStorage) and is sent only to GitHub.

Need a token? The publisher commits straight to your repo via the GitHub API, so it needs a personal access token with repo write access.
  1. Open github.com/settings/tokens/new (scope & name are pre-filled).
  2. Confirm the repo scope is checked, pick an expiration, and click Generate token.
  3. Copy the ghp_… value and paste it below. You won't be able to see it again on GitHub.
Prefer fine-grained tokens? Create one scoped to the blog repo with Contents: Read and write.
